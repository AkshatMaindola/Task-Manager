Task Management Web App

A simple Task Management Web Application built using Spring Boot. This project allows users to perform basic CRUD (Create, Read, Update, Delete) operations on tasks.

Features
Create new tasks
View all tasks
Update existing tasks
Delete tasks
REST API based architecture

Technologies Used
Java
Spring Boot
Spring Data JPA
MySQL
Maven
Purpose

This project was created to practice Spring Boot development and understand how CRUD operations work with a database.
